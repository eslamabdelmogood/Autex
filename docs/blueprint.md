# **App Name**: Industrial Sentinel

## Core Features:

- Machine Connection & Monitoring: Establish a secure connection with industrial machinery via Web Serial API to continuously stream and monitor sensor data.
- Real-time Monitoring Dashboard: Visualize live incoming sensor data, machine status, and key performance indicators in an intuitive, dynamic dashboard.
- AI-Powered Anomaly Detection: An AI tool leveraging generative AI (e.g., Gemini) to analyze streaming sensor data, reason about unusual patterns, detect anomalies, and proactively identify potential equipment malfunctions.
- Configurable Alert System: Allow users to define and customize thresholds for various sensor parameters, triggering immediate alerts when monitored values exceed predefined limits.
- Firebase Data Logging: Automatically store all raw sensor readings, machine states, and alert events in a Firebase Realtime Database for historical analysis and compliance.
- Alert Management Interface: Provide a centralized interface for users to review, acknowledge, escalate, or dismiss triggered alerts, maintaining a comprehensive incident log.

## Style Guidelines:

- Color scheme: Dark theme, reflecting vigilance and data focus. Primary color: Deep blue-violet (#17174C), evoking reliability and advanced technology. Background color: Very dark charcoal-blue (#1B1C24), providing a spacious and non-distracting canvas. Accent color: Bright cyan-blue (#33C4FF), highlighting crucial data points, interactive elements, and alerts.
- Body and headline font: 'Inter' (sans-serif), chosen for its exceptional legibility, modern aesthetic, and suitability for data-rich displays and a professional, objective tone.
- Utilize a consistent set of clean, vector-based line icons that are clearly recognizable and representative of industrial components, sensor types, alert states, and data visualizations. Icons should be functional and integrate seamlessly with the dark theme.
- A responsive dashboard-style layout emphasizing clarity and immediate access to critical information. Key performance indicators and active alerts should be prominently displayed. Data visualizations (charts, gauges) should be well-organized and easy to interpret, providing an at-a-glance overview of machine health.
- Subtle, non-disruptive animations for data updates, alert transitions, and dashboard navigation. Focus on smooth transitions that enhance understanding of data flow and system changes without causing distraction.
